# MikuMikuWorld
世界計劃用譜面編輯/預覽工具

## 必要環境：
- 64bit的Windows 10或以上¹
- [Microsoft Visual C++可再發布](https://aka.ms/vs/17/release/vc_redist.x64.exe)
- GPU對應的OpenGL 3.3和最新的驅動程式

1：更早以前Windows版本可能可以運行，但未經測試。

## 功能一覧：
- 輸出Sliding Universal Score（\*.sus）檔案。

- 可調整BPM、拍子、速度。
- 最大可分割成1920分音符。
- 可製作/使用已儲存的音符組合。
- 可自定義鍵盤快捷鍵。
